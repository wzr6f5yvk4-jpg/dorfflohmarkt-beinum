# Dorfflohmarkt Beinum

Dies ist die öffentliche Website für den Dorfflohmarkt in Salzgitter-Beinum.

Die Seite zeigt eine Übersicht der Verkaufsstellen sowie eine interaktive Karte mit den einzelnen Stationen. Besucherinnen und Besucher können die Website über einen QR-Code aufrufen und sich direkt auf dem Smartphone orientieren.

## Inhalt

* Interaktive Karte mit Verkaufsstellen
* Übersicht aller aktiven Stationen
* Infostelle an der Feuerwehr Beinum
* Hinweise zu Kuchenverkauf, Grill und Getränken
* Pflege der Verkaufsstellen über eine einfache CSV-Datei

## Dateien

### `index.html`

Enthält die komplette Website mit Layout, Karte und Logik.

### `staende.csv`

Enthält die Liste der Verkaufsstellen und der Infostelle.
